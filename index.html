심화 웹 개발

  • 내용 : React.js, vue.js, next.js, spring 을 활용하여 웹 서비스 구축하고 배포하기
  • 학습 목표 : 여러 프레임워크로 웹을 구현한 뒤 자신만의 API서버를 구축하고 배포하기

  • 담당 : 김지호, 박준호

심화 앱 개발

  • 내용 : 심화 Dart 및 Flutter 애니매이션 을 활용하여 앱 만들고 할 수 있으면 배포하기
  • 학습 목표 : 고급 Dart 문법을 활용하여 isolate, async/await 구현 및 API서버 구축하여 활용하기

  • 담당 : 이제석

심화 게임 개발

  • 내용 : Unity 3D 게임 만들고 배포하기
  • 학습 목표 : 3차원 좌표계 이해 및 알고리즘 구현하기

  • 담당 : 이원준
